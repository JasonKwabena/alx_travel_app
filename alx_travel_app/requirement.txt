# Add files to git
git add .
git commit -m "Initial project setup with Django, DRF, Swagger, and Celery"

python manage.py migrate
python manage.py runserver


Now you can access:

    Django admin at http://localhost:8000/admin/

    Swagger docs at http://localhost:8000/swagger/

    API endpoint at http://localhost:8000/api/hello/

To start Celery worker (in a new terminal):
celery -A core worker --loglevel=info

To start RabbitMQ (if not running as service):
rabbitmq-server
